Automated Retinal Image Analyzer

ARIA is software written for MATLAB to detect and measure retinal blood vessels in both fluorescein angiograms and fundus photographs.

It was developed by Pete Bankhead and is described in the PLOS ONE paper Fast retinal vessel detection and measurement using wavelets and edge location refinement.
